# TalentHub Design System\n\n## 🎨 Unified Color System\n\n### Semantic Tokens (ALWAYS USE THESE)\n```css\n/* Primary Brand */\n--primary: 262 83% 58%        /* Purple brand color */\n--primary-foreground: 0 0% 98%\n\n/* Backgrounds */\n--background: 0 0% 100%       /* Main page background */\n--card: 0 0% 100%            /* Card backgrounds */\n--muted: 240 4.8% 95.9%      /* Subtle backgrounds */\n\n/* Text */\n--foreground: 240 10% 3.9%   /* Primary text */\n--muted-foreground: 240 3.8% 46.1% /* Secondary text */\n\n/* Interactive */\n--border: 240 5.9% 90%       /* Borders */\n--accent: 240 4.8% 95.9%     /* Hover states */\n--success: 142 76% 36%       /* Success states */\n--destructive: 0 84.2% 60.2% /* Error states */\n```\n\n## 🧱 Component Standards\n\n### Buttons\n```tsx\n// ✅ CORRECT - Use semantic variants\n<Button variant=\"default\">Primary Action</Button>\n<Button variant=\"outline\">Secondary Action</Button>\n<Button variant=\"ghost\">Subtle Action</Button>\n<Button variant=\"destructive\">Delete</Button>\n<Button variant=\"success\">Confirm</Button>\n\n// ❌ WRONG - Hardcoded colors\n<button className=\"bg-purple-500 text-white\">Bad Button</button>\n<button style={{backgroundColor: '#8b5cf6'}}>Worse Button</button>\n```\n\n### Cards\n```tsx\n// ✅ CORRECT - Semantic card styling\n<Card variant=\"default\" padding=\"default\">\n  <CardHeader>\n    <CardTitle>Card Title</CardTitle>\n    <CardDescription>Card description</CardDescription>\n  </CardHeader>\n  <CardContent>\n    Content here\n  </CardContent>\n</Card>\n\n// ❌ WRONG - Manual styling\n<div className=\"bg-white border border-gray-200 rounded-lg p-6 shadow-sm\">\n  Manual card styling\n</div>\n```\n\n### Layout Spacing\n```tsx\n// ✅ CORRECT - Consistent spacing scale\nclassName=\"space-y-6\"     // Section spacing\nclassName=\"gap-4\"         // Grid gaps\nclassName=\"p-6\"           // Card padding\nclassName=\"px-4 py-8\"     // Page padding\n\n// ❌ WRONG - Random spacing\nclassName=\"mt-3 mb-7 ml-2\" // Inconsistent\nclassName=\"p-5\"            // Off-scale\n```\n\n## 🌓 Dark/Light Mode Rules\n\n### ✅ DO THIS\n```tsx\n// Use semantic tokens\nclassName=\"bg-background text-foreground\"\nclassName=\"bg-card border-border\"\nclassName=\"text-muted-foreground\"\nclassName=\"hover:bg-accent\"\n```\n\n### ❌ NEVER DO THIS\n```tsx\n// Hardcoded colors\nclassName=\"bg-white text-black dark:bg-black dark:text-white\"\nclassName=\"bg-gray-100 dark:bg-gray-800\"\nstyle={{color: '#000000'}}\n```\n\n## 🎯 Typography Scale\n\n```tsx\n// Headings\nclassName=\"text-3xl font-bold\"     // Hero/Page titles\nclassName=\"text-xl font-semibold\"  // Section titles\nclassName=\"text-lg font-medium\"    // Card titles\n\n// Body text\nclassName=\"text-base\"              // Default body\nclassName=\"text-sm text-muted-foreground\" // Helper text\nclassName=\"text-xs text-muted-foreground\"  // Captions\n```\n\n## 🚫 Anti-Patterns (DON'T DO THESE)\n\n### Color Anti-Patterns\n```tsx\n❌ className=\"bg-purple-500\"        // Use bg-primary\n❌ className=\"text-gray-600\"        // Use text-muted-foreground\n❌ className=\"border-gray-300\"      // Use border-border\n❌ className=\"bg-green-500\"         // Use bg-success\n❌ className=\"bg-red-500\"           // Use bg-destructive\n```\n\n### Layout Anti-Patterns\n```tsx\n❌ className=\"rounded-md\"           // Use rounded-lg (consistent radius)\n❌ className=\"shadow-lg\"            // Use shadow-soft/medium/large\n❌ className=\"p-3\"                  // Use p-4 or p-6 (scale)\n❌ className=\"gap-3\"                // Use gap-4 or gap-6\n```\n\n### Component Anti-Patterns\n```tsx\n❌ <div className=\"bg-white p-4 rounded border\"> // Use <Card>\n❌ <button className=\"bg-blue-500 px-4 py-2\">   // Use <Button>\n❌ Manual hover states                           // Use component variants\n```\n\n## 🎨 CSS Utility Classes\n\n```css\n/* Available utility classes */\n.card-modern     /* Modern card with hover effects */\n.gradient-primary /* Primary brand gradient */\n.glass           /* Glass morphism effect */\n.focus-ring      /* Consistent focus styling */\n```\n\n## 📱 Responsive Patterns\n\n```tsx\n// ✅ Mobile-first responsive\nclassName=\"grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3\"\nclassName=\"px-4 md:px-6 lg:px-8\"\nclassName=\"text-sm md:text-base\"\n\n// ❌ Desktop-first\nclassName=\"grid-cols-3 md:grid-cols-1\" // Wrong direction\n```\n\n## 🔧 Implementation Checklist\n\n- [ ] All colors use semantic tokens\n- [ ] No hardcoded hex/rgb colors\n- [ ] Consistent border radius (rounded-lg/xl)\n- [ ] Proper shadow scale (soft/medium/large)\n- [ ] Focus states on interactive elements\n- [ ] Smooth transitions (transition-colors)\n- [ ] Mobile-first responsive design\n- [ ] Dark mode compatibility\n- [ ] Consistent spacing scale\n- [ ] Typography hierarchy\n\n## 🚀 Quick Migration\n\n### Replace These Immediately\n```tsx\n// Old → New\nbg-white → bg-background\nbg-gray-50 → bg-muted\ntext-black → text-foreground\ntext-gray-600 → text-muted-foreground\nborder-gray-200 → border-border\nrounded-md → rounded-lg\nshadow-sm → shadow-soft\nhover:bg-gray-100 → hover:bg-accent\n```\n\nThis design system ensures visual consistency between landing page and dashboard while maintaining modern SaaS aesthetics.
